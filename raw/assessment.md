# Assessment Requirements

> Source: new-full-stack-dev-assessment.docx from Spotter AI (received 2026-05-08)

## Task
Build a full-stack app using Django and React.

## Deliverables
- Live hosted version (Vercel suggested)
- 3-5 minute Loom walkthrough of app + code
- GitHub repo link

## Objective
App that takes trip details as inputs and outputs:
1. Route instructions with a map
2. ELD log sheets drawn/filled out (multiple sheets for longer trips)

## Inputs
- Current location
- Pickup location
- Dropoff location
- Current Cycle Used (hours)

## Outputs
1. **Map** — showing route + stops/rests (use a free map API)
2. **Daily Log Sheets** — drawn on the log grid, filled out, multiple pages for longer trips

## Assumptions (fixed, do not make configurable)
- Property-carrying driver
- 70 hrs / 8 days cycle
- No adverse driving conditions
- Fueling at least once every 1,000 miles
- 1 hour for pickup, 1 hour for dropoff

## Grading
- Hosted version tested for accuracy
- UI/UX matters — good design can compensate for minor inaccuracies
- $100 bonus on successful completion
- Max 4 days / 16 work hours to complete
